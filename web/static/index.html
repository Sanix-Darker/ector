API & Docker usage

Use the public JSON API from any client, or run ECTOR from GitHub Container Registry.

Public API

GET /api/health Service status + supported languages.
GET /api/examples Curated EN/FR samples.
POST /api/extract Returns parsed shopping request JSON.
POST /api/tokenize Counts words and spaCy tokens.
Example request 
curl -X POST https://ector.sanixdk.xyz/api/extract \
  -H "Content-Type: application/json" \
  -d '{"text":"I want two refurbished laptops, budget 700 usd","lang":"en"}'

Run with Docker

Image is published on GitHub Container Registry for fast deployment.

Pull image 
docker pull ghcr.io/sanix-darker/ector:latest
Run container 
docker run --rm -d --name ector-web \
  -p 8000:8000 ghcr.io/sanix-darker/ector:latest
Compose (recommended) 
cd deploy
docker compose pull
docker compose up -d
